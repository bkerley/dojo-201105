Current Beat

Get the current Miami Internet Beat and output it.

The current beat is machine-readable at: http://miamibeats.org/beats.json

Things to know

Loading the simple HTTP client and JSON parser:

require 'open-uri'
require 'json'

Reading a URL as a string:

ip = open('http://bonzoesc.net/ipecho.php').read

Parsing a string of JSON into a Hash:

tweet = JSON.parse('{"text":"good food", "twitter_user_name":"bonzoesc"}')

Accessing a Hash:

tweet['twitter_user_name'] #=> "bonzoesc"
