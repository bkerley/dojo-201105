Things to know

Reading a file into an array lines:

lines = File.read('file.txt').lines.to_a

Splitting a string on spaces:

words = "aeropress coffee" #=> ["aeropress", "coffee"]

Sorting an array based on some math:

sorted = lines.sort_by{|e| e[1].to_i + e[2].to_i }

Printing an array with a space between each entry:

puts words.join(" ")

Printing an array with each entry on its own line:

puts lines
