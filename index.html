Starting Out

FizzBuzz

For each number from 1 to 100, print 'Fizz' if the number is divisible by 3, 'Buzz' if the number is divisible by 5, 'FizzBuzz' if the number is divisible by both, or the number if divisible by neither.

Bench Racing

Given a list of cars, each with weight and horsepower figures, find the power-to-weight ratio for each, and sort them from highest to lowest.

Four-function Calculator

Make a calculator that supports adding, subtracting, mutiplying, and dividing of two integers: "5 + 3" should return "8".
